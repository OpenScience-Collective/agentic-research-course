# Week 7: Manuscript Preparation and Peer Review

## Learning Objectives

By the end of this session, you will:
- Structure an academic manuscript using IMRAD format
- Write each section with appropriate tense and voice
- Format manuscripts for specific journal requirements
- Use AI for peer review calibration
- Respond to reviewer comments effectively

## Outline

### Part 1: IMRAD Structure (10 min)
- Introduction: funnel from broad to specific, ending with objective
- Methods: enough detail for replication
- Results: findings without interpretation
- Discussion: interpretation, limitations, future directions
- Tense conventions per section

### Part 2: Writing with AI (15 min)
- Section-by-section drafting with `/manuscript-prep`
- Clarity: one idea per sentence, active voice
- Precision: quantify claims, distinguish correlation from causation
- Flow: topic sentences, transitions, consistent terminology
- When AI helps vs when you need to write yourself

### Part 3: Journal Formatting (10 min)
- Using `/manuscript-prep "Nature Neuroscience"` for journal-specific formatting
- LaTeX and BibTeX management
- Submission checklist: title page, abstract, figures, cover letter
- Format conversion: markdown to LaTeX, Word to LaTeX

### Part 4: Peer Review (10 min)
- Using `/paper-review` for self-assessment before submission
- What reviewers prioritize: methods, statistics, reproducibility
- Responding to reviewer comments: agree, partially agree, disagree
- The point-by-point response letter

### Q&A (15 min)

## Key Concepts

- **IMRAD:** Introduction, Methods, Results, and Discussion
- **Submission checklist:** Everything needed for a complete submission package
- **Cover letter:** Your pitch to the editor for why this paper fits the journal
- **Point-by-point response:** Structured response addressing each reviewer comment
