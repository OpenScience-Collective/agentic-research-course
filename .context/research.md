# Agentic Research Course -- Research Notes

## Technical Investigations

### matlab-mcp Integration for Practicum
- matlab-mcp-tools provides 15 MCP tools for MATLAB/EEGLAB integration
- Claude Code can execute MATLAB scripts, manage workspace, extract figure data
- Key for the practicum: run EEGLAB processing steps from Claude Code
- Installation: auto-detecting installer handles MATLAB path and Python version

### HBN Dataset
- Healthy Brain Network: large-scale developmental EEG dataset
- Movie-watching paradigm includes shot changes as natural events
- BIDS-compatible after conversion (relevant to week 9)
- Existing pipeline in HBN_BIDS_analysis repo

### research-skills Plugin Ecosystem
- 6 domains: literature, grants, manuscripts, figures, project management, neuroinformatics
- Free, open source: github.com/neuromechanist/research-skills
- Each week of the course maps to a plugin domain
